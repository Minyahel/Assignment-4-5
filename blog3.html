Every New Tag And Attribute In CSS3

CSS3 has been in development since 1999 and it built upon the features of CSS2 and CSS1. Because of it's modularization, it doesn't have a specific release date but rather different modules were released in different year and even more features are in development. CSS3 has added important modules and updates for important aspects of web design while at the same time it introduced various other features that didn't exist before. Below, I have detailed the changes that it introduced as well as which features it introduced were completely new.

New Selectors

With the arrival of CSS3 a host of new selectors have been introduced that make the styling process a lot easier.
These new changes allow us to single out individual elements based on many criteria. With such a fine degree of selection, our control over our document ever increases. The following are the major selectors that were introduced in CSS3.

Attribute Selectors

The CSS attribute selectors allow us to select any type of element based on the specific attribute that they have. This selector works on any CSS element.
The general syntax is given below.
  • E[attribute="value"]: Elements of E type which attribute is equal to its "value"
These attribute selectors can also work like regular expressions meaning that we can write a regular expression for the value if we want to filter out elements through names or other text attributes.

Position Selectors

These selectors allow us to pick out a specific element from our document based on the position of the specific element in our document. This means we can select an element by specifying which type it is and it's number as a child of it's parent element.
The general syntax of these position selectors is given below.
  • :root -> selects the root element
  • E:nth-child(N) -> selects the element of E type that is the Nth child of it's parent
  • E:nth-last-child(N) -> the same as the above but starting from the last element
  • E:nth-of-type(N) -> selects the Nth element of E type in the father element
  • E:nth-last-of-type(N) -> the same as the above but starting from the last element
  • E:first-of-type -> selects an element of E type that comes first in the father element
  • E:last-of-type -> the same as the above but starting from the last element
  • E:only-child -> selects an element of E type that is the only child of its father element
  • E:only-of-type -> selects an element of any type the is the only child of it's type in it's parent

Pseudo-Element selectors

Pseudo-element selectors were available in CSS2 however, a change was brought in CSS3. Pseudo-element selectors allow us to select elements that are not actually elements such as the first letter of a paragraph or the space after a specific element and thus the name pseudo-element. The changes that was brought was the change of the single colon usage(:) to double column usage(::) to distinguish between pseudo-element selectors and pseudo-class selectors.
There are a total of 7 pseudo-element selectors available in CSS3 and they were availabe since CSS1 but since the focus of this post is changes in CSS3, we won't be viewing them here.

Pseudo-Class selectors

Pseudo-class selectors allow us to specify an element based in it's current state and thus they should not be confused with the class we give to groud elements. And thus these selectors allow us to specify an element based on whether it is active or whether the mouse is hovering over it. These were not completely new additions to CSS3 but some changes were added such as the allowing of usage for hover for all elements instead of just anchor tags. The following were changed or added.
  • :hover -> specifies what an element will do when the mouse hovers over it
  • :focus -> specifies what an input element will do when it is activated and ready to receive keyboard input
  • :active -> specifies what an element does when it it currently selected such as a button or current link

Animation

A CSS animation lets an element gradually change from one style to another and adds for the aesthetics of a website. The whole concept of CSS animations was introduced in CSS3. Below are a list of CSS properties that were introduced that allow us to make animations. While we can explicitly write many properties for animation, the animation property can act as a combined attribute allowing us to specify many attributes at once.
  • animation -> specifies animations
  • animation-delay -> specifies a delay before the animation starts
  • animation-direction -> specifies the direction the animation will play in

Background

Background CSS properties were available long before CSS3 but it's functionality grew eventually and CSS3 introduced some key attributes for controlling background.
Below are CSS3 attributes that are new.
  • background-clip: -> specifies where the background should end as in the border-box, padding-box or content-box
  • background-origin: -> specifies wehre the background should start
  • background-size: -> specifies the size of the background

Border

Border were availabe from CSS1 and they are an important part of any design. While the features that were added in were simple, they allow for better design.
Below are the new border properties.
  • border-radius: -> defines the shape of the border corners of an element
  • border-image: -> specifies the usage of an image for a border
In addition to this, there are specifics that control which edge border-radius and border-image should be applied on.

Color

The color property existed before CSS3 but CSS3 added opacity that allows us to control the transparency of any element. It also added functionality for linear gradient and radial gradient. One other major addition related to color is the addition of new color specification formats such as RGBA (with added opacity as alpha), HSL(hue-saturation-lightness) and HSLA.

Flexible Box Layout

Flexible Box or FlexBox for short was a completely new addition to CSS3 that allows us to place elements on screen in a completely new way. Flexbox makes it easier to orient elements either vertically or horizontally and makes it easier to distribute free space among elements.
The major properties associated with flexbox are listed below.
  • display:flex -> sets the display for a selected element to flex
  • flex-direction: -> specifies the direction for a flexbox display
  • flex-wrap: -> specifies whether text should be wrapped or not
  • justify-content: -> specifies the alignment of content
  • align-items: -> specifies how flex items should be aligned

Font Properties

CSS3 didn't get much innovation when it comes to fonts and it added two rather non frequently used properties. The first one is font-size-adjust: which specifies which type of characters to base the font size on and the other one is font-stretch that condenses and stretches fonts.

Multi-column layout Properties

Multi-column layout is a completely new addition to CSS3 that allows the usage of multiple columns to present info.
The following are the properties that are used to control this.
  • column-count: -> specifies the number of columns that will be used
  • column-fill: -> specifies how columns will be filled
  • column-gap: -> specifies the gap between columns
  • column-rule: -> specifies what kind of line will be drawn between columns
  • column-span: -> specifies how many columns an element will span across
Those are the main properties but other shorthand properties can be used as well.

Outline Properties

Outlines are like borders for the element box and they are not new in CSS3, however, it introduced outline-offset that allows to increase the space between an outline and the border edge of an element

Text Properties

Text properties were not a new addition to CSS3, however it introduced properties that allow for greater text control. The properties are listed below.
  • tab-size: -> specifies the length of the tab character
  • text-align-last: -> specifies how the last line of a block is aligned
  • text-decoration-color: -> specifies the color

Transform Properties

The transform properties were new additions to CSS3 and they allow us to apply 2D or 3D transformations to an element such as rotating, scaling and moving elements. The following new tags associated with CSS3 allow us to perform those actions.
  • backface-visibility: -> specifies whether or not the back side of the element will be visible upon transformation
  • perspective: -> defines the perspective from which all child elements of an object are viewed
  • perspective-origin: -> defines the origin (the vanishing point for the 3D space) for the perspective property
  • transform: -> applies a 2D or 3D transformation to an element

Transitions Properties

Transitions were new additions to CSS3 and they were created with the aim of changing property values of elements smoothly and with a controlled delay. The following tags allow us to perform transitions. The transition properties however, have to be applied in junction with pseudo-elements.
  • transition: -> this is a shorthand property that allows us to control which property of an element changes and the delay it takes

Visual Formatting Properties

Visual formatting properties apply to the kind of properties of an element that apply to an element's display and position within the document. Not many additions have been made to this property, however, the new changes are listed below.
  • overflow-x: -> specifies how content is dealt with if it overflows in the x direction
  • overflow-y: -> specifies how content is dealt with if it overflows in the y direction
  • resize: -> specifies whether or not an element is resizable by the user
  • box-shadow: -> allows us to apply a shadow to the element's box
  • box-sizing: -> alters the default CSS box-model
